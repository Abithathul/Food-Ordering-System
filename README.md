**FOOD ORDERING AND MANAGEMENT SYSTEM**

Developed a full-stack web application for online food ordering and restaurant management. The system allows users to browse menus, place orders, manage carts, and track order status efficiently. Implemented responsive user interfaces using HTML, CSS, and JavaScript, with backend functionality and database integration for secure data management. Designed features such as user authentication, order management, and admin controls to enhance the overall user experience and operational efficiency.
